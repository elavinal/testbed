# Testbed
## Getting started
### Dependencies
- `docker-ce`
- `docker-compose-plugin`
- `j2cli`
- `make`

### Known issues
Docker version 5:25.0.0 has [a bug](https://github.com/moby/moby/issues/47120) that prevent running the testbed. Use a different version (5:25.0.1 or higher, or 5:24.* or lower).
